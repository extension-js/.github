# Support

Use the right channel to get help faster.

## Questions and usage help

Use these channels for setup help, troubleshooting, migration questions, and product discussion:

- [GitHub Discussions](https://github.com/extension-js/extension.js/discussions)
- [Discord](https://discord.gg/v9h2RgeTSN)

## Bug reports

If you found a reproducible bug, open an issue in the repository that owns the problem:

- [Extension.js issues](https://github.com/extension-js/extension.js/issues)
- [Documentation issues](https://github.com/extension-js/extension.js.org/issues)
- [Examples issues](https://github.com/extension-js/examples/issues)

## Security reports

Please do not file public issues for vulnerabilities.

Follow the private reporting instructions in the [security policy](https://github.com/extension-js/.github/blob/main/SECURITY.md).
